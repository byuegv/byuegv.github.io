常用bash命令 - 2

cat

cat -n 显示行号 cat -s 不显示重复的换行 cat -A 显示行尾为$，Tab 为^I等

sort

sort -b 从第一个非空白字符开始排序，默认从行首开始排序 sort -n 给予字符串数值的排序 sort -r 反序 sort -k field1,field2 对从field1到field2之间的字符进行排序 sort -t 自定义分隔符，默认由空格或制表符分隔 

ls -l /usr/bin | sort -nr -k 5 | head
#按照第三个域的第5个字符排序
#指定偏移量
ls -al ./ | sort -r -k 3.5 | head
sort -t ':'  --key=1,1 --key=2n example.txt

uniq

uniq file 删除任意重复行 uniq -c 输出重复行并计数 uniq -d 只输出重复行 uniq -D 只输出不重复的行 uniq -i 忽略大小写 uniq -f n 忽略开头的n个字段，字段由空格隔开 uniq -u 只输出独有的文本行 uniq -sn 忽略开头的n个字符

cut

切片和切块
