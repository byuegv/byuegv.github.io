常用bash命令-1

[TOC] Note:bash cmd支持的通配符位* [] ?

  1. ln 链接文件或目录
    > 硬链接：引用文件在文件系统中的物理索引(inode)，移动或删除原文件硬链接不会被破坏，当 > 对文件的索引都被删除，即硬链接数为0时文件被删除。只能用于同种文件系统。
    > 软链接：符号链接，删除软链接不会删除文件本身。可以用于不同的文件系统。
ln -s src dit #创建一个软链接
ln src dit #创建一个硬链接

  1. chown 修改权限
    > u:user g:group o:other a:all > +:add -:unset r:4 w:2 x:1

  2. chown chgrp 修改文件所有者和用户组
    > chown [option] [owner]:[group] file.. > chgrp [option] group file..

  3. setuid setgid 设置用户和权限位
    > 设置了权限位则运行任何人以文件所属用户或用户组的权限执行改文件
    > chown u+s file or chown 4744 file or chown 2755 file or chown 0733 file
    > 进行设置或取消权限位
    > stat file 查看文件信息
